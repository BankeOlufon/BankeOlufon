# 🔒 Windows Privilege Escalation Lab (Local Takeover)

## 📌 Project Overview
An offensive security lab simulating **local privilege escalation** on a Windows environment.  
This project demonstrates how attackers move from restricted user accounts to full SYSTEM/Administrator control by exploiting misconfigurations.

---

## 🎯 Core Areas Covered
- Local Privilege Escalation Techniques  
- Windows Misconfiguration Exploitation  
- Enumeration & Post-Exploitation  
- OSCP PrivEsc Methodology  

---

## 🛠️ Technical Tasks & Functions Performed

### Vulnerable Windows Setup
- Deployed a Windows 10/Server VM with intentional misconfigurations.  
- Left backup credentials exposed or misconfigured services for exploitation.  

### Exploitation Workflow
- Gained initial access as a restricted user.  
- Ran enumeration scripts to identify privilege escalation vectors.  
- Exploited misconfigurations to elevate privileges to SYSTEM/Administrator.  

### Validation & Analysis
- Verified escalation by executing privileged commands.  
- Documented exploitation chain for OSCP-style reporting.
