# 🎮 P5.js Game Development Project

## 📌 Project Overview
A creative coding project built with **p5.js**, showcasing interactive gameplay mechanics.  
The game features a character navigating a scrolling background of trees, avoiding obstacles, and managing a limited number of lives.  
This project demonstrates practical skills in JavaScript-based creative coding, collision detection, and game loop design.

---

## 🎯 Core Areas Covered
- Game Loop & State Management  
- Collision Detection & Player Lives System  
- Background Rendering & Animation  
- Creative Coding with p5.js  

---

## 🛠️ Technical Tasks & Functions Performed

### Game Environment Setup
- Created a scrolling background of trees to simulate movement.  
- Designed obstacles that spawn dynamically across the canvas.  

### Player Character & Controls
- Implemented keyboard controls for character movement.  
- Added collision detection between character and obstacles.  

### Lives & Game Over Logic
- Configured a lives system (3 maximum crashes before game over).  
- Displayed visual feedback when lives were lost.  
- Triggered game-over state after exceeding crash threshold.  

### Visuals & Interactivity
- Used p5.js drawing functions to render sprites and environment.  
- Applied animation loops for smooth gameplay experience.
