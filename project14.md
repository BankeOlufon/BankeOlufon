# 📡 Arduino Secure RFID Door Lock System

## 📌 Project Overview
Designed and implemented a secure RFID-based door lock system using Arduino microcontrollers.  
This project demonstrates physical access control integration with firmware hardening techniques to prevent unauthorized entry and protect against firmware extraction attacks.

---

## 🎯 Core Areas Covered
- Physical Access Control via RFID  
- Embedded Firmware Security & Lock Bit Protections  
- Hardware Integration & Circuit Design  
- Cyber-Physical Security Assessment  

---

## 🛠️ Technical Tasks & Functions Performed

### System Design
- Built an Arduino-based RFID authentication system using standard RFID reader modules.  
- Integrated servo motor control to physically lock/unlock doors upon successful authentication.  

### Firmware Security
- Programmed authentication logic to validate RFID tags against secure stored values.  
- Hardened firmware by configuring ATmega lock bits to prevent unauthorized flash memory reads.  
- Ensured system resilience against physical side-channel attacks.  

### Threat Modeling
- Identified risks of cloned RFID tags and firmware extraction.  
- Proposed mitigations including lock bit programming and encrypted tag validation.  
- Documented security assessment of physical access control systems.  

---

## 🔐 Security Outcomes
- Delivered a functional RFID-based door lock system.  
- Hardened firmware against unauthorized extraction and reverse engineering.  
- Modeled cyber-physical risks in access control environments.
