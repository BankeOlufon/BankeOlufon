# 📡 Smart Accessibility Navigation Node (Cyber-Physical IoT)

## 📌 Project Overview
Designed and engineered an intelligent navigation stick for visually impaired individuals.  
The system leverages ultrasonic, PIR, and water sensors to deliver real-time hazard alerts through haptic and auditory feedback.

---

## 🎯 Core Areas Covered
- Multi-Sensor Data Fusion 
- Edge Feedback Mechanisms (Haptic & Auditory)  
- Cyber-Physical Threat Modeling & Security Assessment  

---

## 🛠️ Technical Tasks & Functions Performed

### Sensor Integration
- Implemented ultrasonic time-of-flight for proximity tracking.  
- Integrated PIR sensor for thermal/motion detection.  
- Deployed resistive sensor grid to detect pooling water hazards.  

### Feedback Mechanisms
- Engineered zero-latency feedback loop translating sensor alerts into PWM signals.  
- Drove auditory buzzers to signal hazard severity.  

### Threat Modeling
- Identified risks of signal spoofing and telemetry manipulation.  
- Proposed acoustic jittering protocol to mitigate ultrasonic jamming.  
- Recommended burning MCU lock bits to prevent firmware extraction.  

---

## 🔐 Security Outcomes
- Delivered real-time hazard alerts for visually impaired navigation.  
- Modeled cyber-physical risks impacting human safety.  
- Proposed mitigations for sensor spoofing and firmware integrity.  

---
