# 🌐 Multi-Cloud Security Posture (AWS + Azure + Wiz/Trend Micro)

## 📌 Project Overview
Built a mock corporate infrastructure across two different cloud providers (AWS and Azure) and connected them both to a single, centralized security dashboard (Wiz / Trend Micro).  
This project demonstrates the ability to monitor an entire company's cloud footprint from one screen and identify dangerous security misconfigurations before attackers do.

---

## 🎯 Core Areas Covered
- Multi-Cloud Infrastructure Setup (AWS + Azure)  
- Centralized Security Dashboard Integration (Wiz / Trend Micro)  
- Security Baseline Auditing & Misconfiguration Simulation  
- Attack Path Visualization & Toxic Combination Detection  

---

## 🛠️ Technical Tasks & Functions Performed

### Multi-Cloud Setup
- Built infrastructure in AWS (EC2 instance, S3 bucket) and Azure (VMs, blob storage).  
- Connected both cloud accounts to Wiz / Trend Micro Cloud One using API access keys.  

### Simulated Security Mistakes
- Exposed storage containers in AWS and Azure to public internet access.  
- Configured open firewall ports (SSH, RDP) to simulate risky exposure.  
- Created an over-privileged identity key attached to a basic application.  

### Toxic Combination Detection
- Used Wiz / Trend Micro dashboard to correlate multiple misconfigurations into a single attack path.  
- Visualized how open ports, unpatched servers, and over-privileged keys could lead to sensitive data compromise.  
- Prioritized remediation by fixing the root firewall rule to break the attack chain.
