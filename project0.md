# 🔒 Active Directory Domain Pivot Lab (Network Conquest)

## 📌 Project Overview
An offensive security lab simulating **Active Directory attacks and lateral movement**.  
This project demonstrates how attackers pivot from a compromised workstation to full domain controller takeover.

---

## 🎯 Core Areas Covered
- Active Directory Attacks & Credential Theft  
- Lateral Movement Techniques  
- Domain Controller Compromise  
- OSCP AD Methodology  

---

## 🛠️ Technical Tasks & Functions Performed

### Lab Setup
- Built a small network with attacker machine, Windows workstation, and Domain Controller.  
- Configured AD environment to simulate corporate infrastructure.  

### Exploitation Workflow
- Gained admin control of workstation machine.  
- Extracted credential hashes from system memory.  
- Used stolen credentials to log into Domain Controller.  

### Validation & Analysis
- Verified domain controller compromise.  
- Documented lateral movement chain for OSCP-style reporting.
